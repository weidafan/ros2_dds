Detailed Description

Indicates if the samples are from a live dds::pub::DataWriter or not.

For each instance, the middleware internally maintains an instance state. The instance state can be:

  • dds::sub::status::InstanceState::alive() indicates that (a) samples have been received for the instance, (b) there are live dds::pub::DataWriter entities writing the instance, and (c) the instance has not been explicitly disposed (or else more samples have been received after it was disposed).
  • dds::sub::status::InstanceState::not_alive_disposed() indicates the instance was explicitly disposed by a dds::pub::DataWriter by means of the dispose operation.
  • dds::sub::status::InstanceState::not_alive_no_writers() indicates the instance has been declared as not-alive by the dds::sub::DataReader because it detected that there are no live dds::pub::DataWriter entities writing that instance.

The precise behavior events that cause the instance state to change depends on the setting of the OWNERSHIP QoS:

  • If OWNERSHIP is set to dds::core::policy::OwnershipKind_def::EXCLUSIVE, then the instance state becomes dds::sub::status::InstanceState::not_alive_disposed() only if the dds::pub::DataWriter that "owns" the instance explicitly disposes it. The instance state becomes dds::sub::status::InstanceState::alive() again only if the dds::pub::DataWriter that owns the instance writes it.
  • If OWNERSHIP is set to dds::core::policy::OwnershipKind_def::SHARED, then the instance state becomes dds::sub::status::InstanceState::not_alive_disposed() if any dds::pub::DataWriter explicitly disposes the instance. The instance state becomes dds::sub::status::InstanceState::alive() as soon as any dds::pub::DataWriter writes the instance again.

The instance state available in the dds::sub::SampleInfo is a snapshot of the instance state of the instance at the time the collection was obtained (i.e. at the time read or take was called). The instance state is therefore the same for all samples in the returned collection that refer to the same instance.
