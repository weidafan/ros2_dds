Creates a multidimensional array.

For example, to create the following IDL array:

double x[10][20];

We can use this code:

std::vector<uint32_t> dim(2);
dim[0] = 10;
dim[1] = 20;
ArrayType my_array(primitive_type<double>(), dim.begin(), dim.end());
