Detailed Description

QoS policies supported by a DDS_Publisher entity.

You must set certain members in a consistent manner:

length of DDS_GroupDataQosPolicy::value <= DDS_DomainParticipantResourceLimitsQosPolicy::publisher_group_data_max_length

length of DDS_PartitionQosPolicy::name <= DDS_DomainParticipantResourceLimitsQosPolicy::max_partitions

combined number of characters (including terminating 0) in DDS_PartitionQosPolicy::name <= DDS_DomainParticipantResourceLimitsQosPolicy::max_partition_cumulative_characters

If any of the above are not true, DDS_Publisher_set_qos and DDS_Publisher_set_qos_with_profile will fail with DDS_RETCODE_INCONSISTENT_POLICY and DDS_DomainParticipant_create_publisher will return NULL.
