Detailed Description

Entry created when a Topic object discovered.

Data associated with the built-in topic builtin.TopicBuiltinTopicDataTypeSupport.TOPIC_TOPIC_NAME. It contains QoS policies and additional information that apply to the remote com.rti.dds.topic.Topic.

Note: The builtin.TopicBuiltinTopicData built-in topic is meant to convey information about discovered Topics. This Topic's samples are not propagated in a separate packet on the wire. Instead, the data is sent as part of the information carried by other built-in topics (builtin.PublicationBuiltinTopicData and builtin.SubscriptionBuiltinTopicData). Therefore TopicBuiltinTopicData DataReaders will not receive any data.
