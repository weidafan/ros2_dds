Detailed Description

Entry created when a com.rti.dds.publication.DataWriter is discovered in association with its Publisher.

Data associated with the built-in topic builtin.PublicationBuiltinTopicDataTypeSupport.PUBLICATION_TOPIC_NAME. It contains QoS policies and additional information that apply to the remote com.rti.dds.publication.DataWriter the related com.rti.dds.publication.Publisher.
