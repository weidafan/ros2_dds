RTI Routing Service Documentation

RTI Routing Service is an RTI Connext Messaging component that supports the integration and scaling of disparate and geographically dispersed systems.

Using off-the-shelf or custom developed plugins, RTI Routing Service supports the definition of data transformations and adapters that can interface to multiple protocols in addition to DDS, such as JMS or legacy code written to the network socket API.

You can learn how to use RTI Routing Service through the User's Manual and the Getting Started Guide.

A reference to the API can be found here:

  • RTI Routing Service Adapter API
  • RTI Routing Service API.
