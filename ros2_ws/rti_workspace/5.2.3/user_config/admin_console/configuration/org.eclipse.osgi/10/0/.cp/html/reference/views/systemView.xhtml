System View

Connected Domains
The list of domains currently used for communication is displayed at the top. To join additional domains or leave current domains, click on the link "Connected to domains" to open the Preferences panel.

System Summary Table
This table provides an overview of what is running in the distributed system and whether any of these components need your attention.
  • Hosts - The number of hosts connected to Admin Console.

  • Processes - The number of processes discovered by Admin Console.

  • Processes Reporting Errors - The number of processes with an error health state.

  • RTI Services - The number of RTI services discovered by Admin Console.

  • RTI Services Reporting Errors - The number of RTI services that are reporting errors.


Hosts Tab
This table provides an overview of each host that is communicating with Admin Console.
  • Hosts - Name/IP of the hosts.

  • Health - Aggregate of the health value of all the services / processes recognized by Admin Console on this host.

  • Up-Time - The time since the system has started until now.

  • CPU (%) - Current CPU usage of this host. This value is retrieved using "/proc/stat" on Linux systems and "GetSystemInfo" on Windows systems.

  • Used Memory - Current memory usage of this host. This value is retrieved using "sysinfo" on Linux systems and "GlobalMemoryStatusEx" on Windows systems.

  • Free Memory - Current free memory available on this host. This value is retrieved using "sysinfo" on Linux systems and "GlobalMemoryStatusEx" on Windows systems.

  • Used Swap Memory - Current swap memory usage of this host. This value is retrieved using "sysinfo" on Linux systems and "GlobalMemoryStatusEx" on Windows systems.

  • Free Swap Memory - Current free swap memory available on this host. This value is retrieved using "sysinfo" on Linux systems and "GlobalMemoryStatusEx" on Windows systems.


System Log Tab
The System Log is a table that shows log messages (provided by Distributed Logger) aggregated from all of the processes in the entire distributed system.
