Health View
Overview
The health view presents a tabular view of the reasons for the health state of the selected system entity. Each reason is represented in a row of the table. The view's icon will change to represent the overall state.
  • Health - This is the health state for the row's reason.
  • Reason - This is the title of the health reason. Clicking on this cell will open a view with more details.
  • Description - A detailed description of the health reason.
Toolbar
  • Link with selection - Toggling this button will link/unlink this view to/from displaying information about the current selection. One way to use this feature is to select something and then unlink this view from the selection. Then, a new instance of this view can be created and a populated with a new selection. Data can then be compared between the first and second selection.
