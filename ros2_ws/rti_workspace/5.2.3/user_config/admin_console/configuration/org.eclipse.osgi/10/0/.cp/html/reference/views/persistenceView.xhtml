Persistence Service View

The Persistence Service View is a Process View with additional controls specific to Persistence Service. Refer to the Process View for details on the common displays.

Admin Console provides basic administration capabilities for Persistence Service. You can enable, disable, and shutdown the Persistence Service remotely. Admin Console also displays the distributed logs from the Persistence Service. To set up Persistence Service for Admin Console, verify administration and distributed logging are enabled in the configuration file. See the Persistence Service secition of the RTI Core Libraries and Utilities User's Manual for details on how to enable administration and distributed logging.
