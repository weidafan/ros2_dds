Toolbar
Preferences  
This button display the Preferences Dialog.

Open Configuration  
This button allows you to open an XML file so that it can be edited.

Physical View  
This button shows or hides the Physical View.

DDS Logical View  
This button shows or hides the DDS Logical View.

Processes View  
This button shows or hides the Processes View.

RTI Internal Topics Filter  
This button turns filtering for RTI internal topics on or off. When on (the default), DDS topics that are used to control or monitor RTI services or functionality are hidden from view. You can turn this filter off but this should only be done in cases where you are trying to debug RTI internal topics (or to gain a better understanding of the data types or quality of service being employed by these topics).

Match Analyses View  
This button creates a new Match Analyses View.

Health View  
This button creates a new Health View.

DDS Data Type View  
This button creates a new DDS Data Type View.

DDS QoS View  
This button creates a new DDS QoS View.

Open Data Visualization Perspective
This button changes the perspective of the Admin Console from the default of Administration to Data Visualization.

Sample Inspector
This button opens a new Sample Inspector view. This and other Data Visualization views are covered in more depth in the Data Visualization section of the help.

Sample Log
This button opens a new Sample Log view. This and other Data Visualization views are covered in more depth in the Data Visualization section of the help.

Time Chart
This button opens a new Time Chart view. This and other Data Visualization views are covered in more depth in the Data Visualization section of the help.

Back / Forward buttons  
These arrows help you navigate to Admin Console views that have been previously opened.

Open Perspective
Pushing this button displays a dialog where different perspectives can be chosen to be active. Note that only Administration and Data Visualization are supported. The other perspectives come as part of the platform (Eclipse) upon which the Admin Console application is built.

Administration Perspective
Pushing this button changes the current perspective to be Administration.

Data Visualization Perspective
Pushing this button changes the current perspective to be Data Visualization. Note that this button is not displayed until after the Data Visualization perspective is shown once. There is another button to invoke it directly as described above.
