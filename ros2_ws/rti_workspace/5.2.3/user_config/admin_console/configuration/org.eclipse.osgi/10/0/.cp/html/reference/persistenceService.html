Persistence Service Details

Admin Console provides basic administration capabilities for Persistence Service. You can enable / disable persistence service remotely. Admin Console also displays the distributed logs from Persistence Service. To setup Persistence Service for Admin Console, verify administration and distributed logging is enabled in the Persistence service, and also make sure the domain used by administration is joined by the Admin Console. See Persistence Service's user manual for details on how to enable Administration and Distributed Logging, and see Admin Console setup for details on how to join addditional domains. Additional support for persistence service will be added in a future release.
