Console Log View
Overview
The console log presents a tabular view of the log messages produced by Admin Console itself. It is normally not necessary to display this view, but it can be brought up if there's a suspicion that the process is having trouble.
  • Time - The time the log message was written.
  • Level - The level of the message.
  • Logger Name - The name of the class which wrote the message.
  • Message - The message.
Toolbar
This view's toolbar contains a button to open the log file, a button to display the log preferences , and buttons to minimize and maximize the view.
