XML Editor
The XML Editor provides a full-featured graphical editor for XML files. If a Routing Service configuration file triggered the opening of this editor it will be augmented with a vertical toolbar along it's left side (as shown below).
The toolbar contains buttons to:
  • Save - Save the configuration file to disk.
  • Save As - Save the configuration file to disk with a different name than the current one.
  • Refresh - Get a fresh copy of the configuration from the Routing Service.
  • Send - Send the configuration to the Routing Service.
