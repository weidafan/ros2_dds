Replay Service View
The Replay Service View is a Process View with additional controls specific to the Replay Service. Refer to the Process View for details on the common displays.

Admin Console provides basic administration capabilities for Replay Service. You can enable, disable, pause, resume, and shutdown Replay Service remotely. Note that to request the service to shutdown, it must be idle (paused). Admin Console also displays the distributed logs from Replay Service. To setup Replay Service for Admin Console, verify administration and Distributed Logging is enabled in the Replay Service.
See Replay Service's user manual for details on how to enable Administration and Distributed Logging.
